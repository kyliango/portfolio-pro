EcoTrack Dashboard

Une application web interactive conçue pour sensibiliser les étudiants universitaires à leur empreinte carbone quotidienne.

Approche UX/UI

Création de personas étudiants pour cibler les besoins précis. Design émotionnel utilisant des teintes vertes apaisantes (psychologie des couleurs) et des micro-interactions satisfaisantes lors de l'accomplissement d'objectifs (gamification).

Solution Apportée

Création d'un tableau de bord gamifié. Utilisation de graphiques interactifs (Chart.js) pour visualiser les progrès et système de badges pour récompenser les actions écologiques.

Stack Technique

Figma (Maquettes) Next.js Tailwind CSS Chart.js
